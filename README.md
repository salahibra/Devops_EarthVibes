# EarthVibesApp
## Pre-requirements: 
**node : version>=20**
**npm**
## install packages:
```
npm i
```
## install angular-cli globally
```
npm i -g @angular/cli
```
## install firebase tools globally
```
npm i -g firebase-tools
```
## login to firebase account
```
firebase login
```
## install angular-fire library
```
npm i @angular/fire --legacy-peer-deps
```
